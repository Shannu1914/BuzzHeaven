# Social Media Project

## Overview
Full-stack social media application with:

- User authentication
- Posts, feeds, likes, comments
- Real-time messaging
- Audio & video calls (WebRTC)
- Admin panel for users, posts, reports

## Project Structure
- `/backend` — Node.js + Express + EJS + Socket.io
- `/frontend` — React / Next.js frontend
- `/mobile` — React Native mobile app
- `/docker` — Docker setup
- `/docs` — API, DB, media, WebRTC, Socket documentation

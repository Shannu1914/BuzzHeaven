# API Documentation

- POST /auth/register
- POST /auth/login

## Users
- GET /users/:id
- PATCH /users/:id
- GET /users/me

## Posts

- POST /posts

- PATCH /posts/:id
- DELETE /posts/:id

## Media

- GET /media/:filename

## Messages

- POST /messages
